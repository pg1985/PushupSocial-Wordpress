Pushup Social is the easiest way to add an online community to your WordPress site. Simply install the plugin to your site, then either add an existing community or create a new one for your site.

== Description ==

Humanize Your Brand

Pushup is the best way to make your website more social. It combines an online community AND social media aggregator into one social hub, directly on your website. Your visitors can now view and interact with social feeds pulled in from Facebook, Twitter and Instagram, as well as curate original content unique to your Pushup community.

This plugin automatically inserts a sleek, customizable bar that sits at the bottom of your site. To get started, simply sign up directly in the plugin's settings upon adding it to your WordPress site.

== Installation ==

=There are two ways you can install the Pushup Social for WordPress plugin; Automatic and Manual Installation.

Automatic Installation

1. From the WordPress Dashboard, go to `Plugins > Add New`.
2. Under Search, type in `Pushup Social` and click Search Plugins.
3. The Pushup Social plugin version 1.5.0 will populate.
4. Click Install Now to install the Pushup Social Plugin.
5. A popup window will ask you to confirm your wish to install the Plugin.
6. Click Proceed or Ok to continue with the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
7. If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.

Manual Installation

1. Click the Download Now button below and download the .zip file to your desktop.
2. Extract the `pushup-social` Plugin folder to your desktop.
3. Read through the `readme` file thoroughly to ensure you follow the installation instructions.
4. With your FTP program, upload the `pushup-social` Plugin folder to the wp-content/plugins folder in your WordPress directory online.
5. From your WordPress dashboard, go to Plugins screen and find the newly uploaded Pushup Social Plugin in the list.
6. Click Activate Plugin to activate it.
